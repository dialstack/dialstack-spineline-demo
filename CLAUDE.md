This application is a vertical SaaS platform for chiropractors that showcases
DialStack embedded voice components, allowing businesses to handle PBX
telephony directly within the platform. Because this is an example, the code
needs to be simple and to-the-point, without fluff. Everything will be well
documented and commented.

Technology choices:

Frontend
- Next.js
- React
- Tailwind CSS
- shadcn/ui
- Material-UI

Authentication & Backend
- NextAuth.js
- PostgreSQL
- bcrypt/bcryptjs

Form Management & Validation
- React Hook Form
- Zod
- @hookforms/resolvers

State Manamgent & Data Fetching
- TanStack Query (React Query)

Development & Build Tools
- TypeScript
- ESLint
- Prettier
- Docker
